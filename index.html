Obstacle Detection

A machine learning application for patients with Parkinson's Disease

It is a project for ECE Design Competition
It uses an machine learning object detection model (Single Shot Detector with MobileNet v2) to detect any obstacles to help patients with Parkinson's disease to be aware of their surrounding.
It only uses a single camera and is currently only available on iOS, although the model is trained with Tensorflow and is able to be used cross-platform.
We end up being the most voted project by the audiences after the final presentation but still didn't win the competition (because we didn't communicate enough with the patients).
